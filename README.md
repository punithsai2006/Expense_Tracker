# Expense Tracker (Python)

A beginner-friendly Python-based Expense Tracker application that allows users to record daily expenses, view monthly summaries, visualize category-wise spending using charts, and export expense data to CSV files.

---

## Features

- Add daily expenses with date, category, amount, and description  
- View monthly expense summary  
- Visualize expenses using bar charts  
- Export all expense data to a CSV report  

---

## Technologies Used

- Python  
- CSV file handling  
- Matplotlib (for data visualization)
